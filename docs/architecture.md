## 1. РһРұР·РҫСҖ СҒРёСҒСӮРөРјСӢ

### 1.1 РқР°Р·РҪР°СҮРөРҪРёРө

Agent Message Bus вҖ” Р»РҫРәР°Р»СҢРҪР°СҸ СҲРёРҪР° СҒРҫРҫРұСүРөРҪРёР№ РҙР»СҸ РҫСҖРәРөСҒСӮСҖР°СҶРёРё AI-Р°РіРөРҪСӮРҫРІ РІ РҝСҖРҫСҶРөСҒСҒРө СҖР°Р·СҖР°РұРҫСӮРәРё. РЎРёСҒСӮРөРјР° РҫРұРөСҒРҝРөСҮРёРІР°РөСӮ СҒСӮСҖСғРәСӮСғСҖРёСҖРҫРІР°РҪРҪСғСҺ РәРҫРјРјСғРҪРёРәР°СҶРёСҺ РјРөР¶РҙСғ Р°РіРөРҪСӮР°РјРё СҮРөСҖРөР· РҝРҫСӮРҫРәРё СҒРҫРҫРұСүРөРҪРёР№ (threads) СҒ РіР°СҖР°РҪСӮРёРөР№ РҙРҫСҒСӮР°РІРәРё.

## 2. РҗСҖС…РёСӮРөРәСӮСғСҖРҪСӢРө РҝСҖРёРҪСҶРёРҝСӢ

| **РҹСҖРҫСҒСӮРҫСӮР°** | РңРёРҪРёРјСғРј РҙРІРёР¶СғСүРёС…СҒСҸ СҮР°СҒСӮРөР№ | РҹСҖРҫСҒСӮРҫСӮР° РҫСӮР»Р°РҙРәРё Рё СҚРәСҒРҝР»СғР°СӮР°СҶРёРё |

| **РқР°РҙС‘Р¶РҪР°СҸ РҙРҫСҒСӮР°РІРәР°** | ACK + Retry + DLQ | Р“Р°СҖР°РҪСӮРёСҸ РҫРұСҖР°РұРҫСӮРәРё |

const agent = await client.registerAgent({

| РҗСҒРҝРөРәСӮ | РўРөРәСғСүРөРө СҒРҫСҒСӮРҫСҸРҪРёРө | РһРұРҫСҒРҪРҫРІР°РҪРёРө |
|--------|-------------------|-------------|
| РҗСғСӮРөРҪСӮРёС„РёРәР°СҶРёСҸ | РһСӮСҒСғСӮСҒСӮРІСғРөСӮ | РўРҫР»СҢРәРҫ Р»РҫРәР°Р»СҢРҪРҫРө СҖР°Р·РІС‘СҖСӮСӢРІР°РҪРёРө |
| РҗРІСӮРҫСҖРёР·Р°СҶРёСҸ | РһСӮСҒСғСӮСҒСӮРІСғРөСӮ | РЎСҶРөРҪР°СҖРёР№ РҫРҙРҪРҫРіРҫ РҝРҫР»СҢР·РҫРІР°СӮРөР»СҸ |

### 10.2 Р РөРәРҫРјРөРҪРҙР°СҶРёРё РҙР»СҸ production

Р•СҒР»Рё СҒРёСҒСӮРөРјР° РұСғРҙРөСӮ СҖР°Р·РІС‘СҖРҪСғСӮР° РҪРө Р»РҫРәР°Р»СҢРҪРҫ:

1. **API Keys** вҖ” РҙРҫРұР°РІРёСӮСҢ Р°СғСӮРөРҪСӮРёС„РёРәР°СҶРёСҺ СҮРөСҖРөР· API РәР»СҺСҮРё

### 12.3 РЈСӮРҫСҮРҪРөРҪРёСҸ Р°СҖС…РёСӮРөРәСӮРҫСҖР° (Epic 1вҖ“6)

РқРёР¶Рө вҖ” Р·Р°С„РёРәСҒРёСҖРҫРІР°РҪРҪСӢРө РәРҫРҪСӮСҖР°РәСӮСӢ Рё СғРәР°Р·Р°СӮРөР»Рё РҙР»СҸ СҖР°Р·СҖР°РұРҫСӮРәРё РҝРҫ СҚРҝР°Рј.

#### РҳРҪСӮРөСҖС„РөР№СҒ С…СҖР°РҪРёР»РёСүР° (Epic 1, packages/core)

Р•РҙРёРҪСӢР№ РәРҫРҪСӮСҖР°РәСӮ РҝРөСҖСҒРёСҒСӮРөРҪСӮРҪРҫСҒСӮРё вҖ” **`MessageBusStorage`** (`packages/core/src/storage/interface.ts`). Р’СҒРө РҫРҝРөСҖР°СҶРёРё РҝСҖРёРҪРёРјР°СҺСӮ `projectId`; СҖРөР°Р»РёР·Р°СҶРёСҸ (Prisma РІ `packages/db` РёР»Рё in-memory РҙР»СҸ СӮРөСҒСӮРҫРІ) РҫРұРөСҒРҝРөСҮРёРІР°РөСӮ РёР·РҫР»СҸСҶРёСҺ РҝРҫ РҝСҖРҫРөРәСӮСғ.

- **Agents:** `listAgents`, `createAgent`, `searchAgents`, `getAgentById`
- **Threads:** `listThreads`, `createThread`, `getThreadById`, `listThreadMessages`, `updateThreadStatus`, `deleteThread`
- **Messages:** `createMessage`, `getMessageById`, `updateMessageStatus`, `findMessages`, `getInboxAndMarkDelivered`, `updateManyMessages`, `updateManyMessagesToStatus`, `deleteManyMessages`

РўРёРҝСӢ РІС…РҫРҙРҫРІ: `CreateAgentInput`, `CreateThreadInput`, `SendMessageInput` вҖ” РІ СӮРҫРј Р¶Рө С„Р°Р№Р»Рө. Р Р°СҒСҲРёСҖСҸСӮСҢ РёРҪСӮРөСҖС„РөР№СҒ СӮРҫР»СҢРәРҫ СҒ СғСҮС‘СӮРҫРј РҫРұСҖР°СӮРҪРҫР№ СҒРҫРІРјРөСҒСӮРёРјРҫСҒСӮРё СҒСғСүРөСҒСӮРІСғСҺСүРёС… СҖРөР°Р»РёР·Р°СҶРёР№.

#### РҡРҫРҪСӮСҖР°РәСӮ RLS helpers (Epic 1, packages/db)

Р’ `packages/db/src/rls.ts` СҚРәСҒРҝРҫСҖСӮРёСҖСғСҺСӮСҒСҸ РҙРІРө С„СғРҪРәСҶРёРё РҙР»СҸ СғСҒСӮР°РҪРҫРІРәРё РәРҫРҪСӮРөРәСҒСӮР° **РІ СҖР°РјРәР°С… РҫРҙРҪРҫР№ SQL-СӮСҖР°РҪР·Р°РәСҶРёРё** (`SET LOCAL` СҮРөСҖРөР· `set_config(..., true)`):

- **`setTenantContext(tx, tenantId)`** вҖ” СғСҒСӮР°РҪР°РІР»РёРІР°РөСӮ `app.tenant_id`
- **`setProjectContext(tx, projectId)`** вҖ” СғСҒСӮР°РҪР°РІР»РёРІР°РөСӮ `app.project_id`

Р’СӢР·СӢРІР°СӮСҢ РІ РҪР°СҮР°Р»Рө СӮСҖР°РҪР·Р°РәСҶРёРё РҙРҫ Р»СҺРұСӢС… СҮСӮРөРҪРёР№/Р·Р°РҝРёСҒРөР№; РҝРҫР»РёСӮРёРәРё RLS (ADR-008) РҫРҝРёСҖР°СҺСӮСҒСҸ РҪР° СҚСӮРё РҝРөСҖРөРјРөРҪРҪСӢРө. РқРө РёСҒРҝРҫР»СҢР·РҫРІР°СӮСҢ РІРҪРө СӮСҖР°РҪР·Р°РәСҶРёРё Prisma.

#### Operational readiness (Epic 6)

РҹРҫ Р·Р°РҝСҖРҫСҒСғ РҝСҖРё СҖРөР°Р»РёР·Р°СҶРёРё Epic 6 СғСӮРҫСҮРҪСҸСӮСҢ:

- **Rate limiting** вҖ” РҫРіСҖР°РҪРёСҮРөРҪРёРө Р·Р°РҝСҖРҫСҒРҫРІ РҝРҫ API (РҪР° СғСҖРҫРІРҪРө РҝСҖРёР»РҫР¶РөРҪРёСҸ РёР»Рё reverse proxy); РҝРҫСҖРҫРіРё Рё СҒСӮСҖР°СӮРөРіРёСҸ (РҝРҫ IP, РҝРҫ СӮРҫРәРөРҪСғ, РҝРҫ endpoint) вҖ” РІ РәРҫРҪС„РёРіСғСҖР°СҶРёРё РёР»Рё ADR.
- **Observability** вҖ” РјРөСӮСҖРёРәРё (СҖР°Р·РҙРөР» 11.2), Р»РҫРіРёСҖРҫРІР°РҪРёРө СҒСӮСҖСғРәСӮСғСҖРёСҖРҫРІР°РҪРҪРҫРө, СӮСҖР°СҒСҒРёСҖРҫРІРәР° РҝСҖРё РјР°СҒСҲСӮР°РұРёСҖРҫРІР°РҪРёРё; health endpoint (`/health` РёР»Рё Р°РҪР°Р»РҫРі) РҙР»СҸ readiness/liveness.
- **Deployment** вҖ” ADR-009 (Kubernetes + Podman); РҝСҖРҫСҶРөСҒСҒ СҒРұРҫСҖРәРё, РјРёРіСҖР°СҶРёРё Р‘Р” РҙРҫ СҒСӮР°СҖСӮР° РҝСҖРёР»РҫР¶РөРҪРёСҸ, РҫСӮРәР°СӮ.
- **Backup** вҖ” СҒСӮСҖР°СӮРөРіРёСҸ РұСҚРәР°РҝРҫРІ PostgreSQL (РҝРөСҖРёРҫРҙРёСҮРҪРҫСҒСӮСҢ, С…СҖР°РҪРөРҪРёРө, РІРҫСҒСҒСӮР°РҪРҫРІР»РөРҪРёРө); СҒРҫРіР»Р°СҒРҫРІР°СӮСҢ СҒ С…РҫСҒСӮРёРҪРіРҫРј (ADR-009).
