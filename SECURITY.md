# Политика безопасности

## Сообщение об уязвимостях

Если вы обнаружили уязвимость в Agent Message Bus, пожалуйста, **не создавайте публичный Issue**.

Предпочтительные способы сообщить о проблеме:

1. **GitHub Security Advisories** (рекомендуется): [Report a vulnerability](https://github.com/bizmedia/amb/security/advisories/new) — сообщение останется конфиденциальным до публикации исправления.
2. **Напрямую мейнтейнерам** — если в репозитории указан контакт для безопасности, воспользуйтесь им.

Опишите, по возможности: шаги воспроизведения, влияние (что можно сделать, эксплуатируя уязвимость), версию/окружение. Мы постараемся ответить в разумный срок и сообщим о плане исправления.

После выпуска фикса мы можем упомянуть вас в благодарностях (если вы не против). Спасибо за ответственный подход.
