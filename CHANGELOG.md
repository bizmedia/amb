## [0.1.0] — 2026-01-28

### Добавлено

- Локальная шина сообщений для оркестрации ИИ-агентов (threads, inbox, ACK, retry, DLQ).
- Next.js приложение: REST API и Dashboard UI.
- TypeScript SDK и MCP-сервер для Cursor.
- Интеграция с PostgreSQL (Prisma).
- Скрипты: seed агентов и тредов, retry-воркер, cleanup, orchestrator.
- Документация: README, QUICKSTART, getting-started, API, архитектура, ADR.
